# ISAM
- Currently using for myCobot280 ElephantRobotics Robotic Arm:
  - Jetson Orin Nano Super: The brain that holds the AI model.
  - Arduino Uno R3: The ultimate safety controller that relays communication signals between the Jetson nano and the Raspberry Pi.
  - Raspberry Pi 4 Model B: embedded in the arm that is able to control the arm.
