# ISAM
- Currently using for myCobot280 ElephantRobotics Robotic Arm:
  - Gazebo Harmonic: Simulator that realistically visualizes robotic arm 
  - ROS2 Jazzy: What controls the Robotic Arm
  - MoveIt 2: plans movements for the robotic arm 
  - Rviz2: Visualizes what ROS2 is doing to the robotic arm
  - Docker: Container that isolates environment of the robotic arm
